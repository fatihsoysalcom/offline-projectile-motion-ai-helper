# Offline Projectile Motion AI Helper

This example demonstrates an offline STEM simulation using Python's standard library. It simulates projectile motion and includes a basic 'AI assistant' that calculates launch angles needed to hit a specific target. This showcases how 'offline AI simulations' can provide interactive problem-solving capabilities without internet access, making STEM education more accessible.

## Language

`python`

## How to Run

Save the code as `main.py`.
Run from your terminal: `python main.py`
